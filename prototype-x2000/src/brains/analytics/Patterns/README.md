# Analytics Brain — Patterns Library

## Purpose

Patterns encode repeatable workflows for common analytics operations. Each pattern captures the complete process from requirements through delivery, including quality gates, stakeholder coordination, and common failure modes.

---

## Pattern Inventory

| Pattern | File | Use Case |
|---------|------|----------|
| Dashboard Design | `dashboard_design_pattern.md` | Designing and building analytics dashboards |
| Metric Definition | `metric_definition_pattern.md` | Defining, validating, and governing metrics |
| Analytics Project | `analytics_project_pattern.md` | Executing end-to-end analytics projects |

---

## When to Use Patterns

- **Dashboard Design Pattern:** When building any new dashboard or redesigning existing ones
- **Metric Definition Pattern:** When introducing new metrics or resolving metric inconsistencies
- **Analytics Project Pattern:** For any ad-hoc or strategic analysis project

| Pattern | Brain Dependencies |
