## 8. Day-of Checklist

### Pre-Event (2 hours before)

- [ ] Venue/platform setup complete
- [ ] A/V tested (projector, microphone, screen sharing)
- [ ] WiFi tested with multiple devices
- [ ] Registration/check-in ready
- [ ] Signage in place (if in-person)
- [ ] Catering confirmed and arriving on time
- [ ] Swag table set up
- [ ] All speakers confirmed and briefed
- [ ] Emergency contacts shared with team
- [ ] Recording setup tested

### During Event

- [ ] Welcome attendees, point to resources
- [ ] Introduce speakers on time
- [ ] Monitor chat/Q&A for questions
- [ ] Take photos/screenshots for social media
- [ ] Post live updates to social media
- [ ] Handle any technical issues promptly
- [ ] Time-keep for speakers

### Post-Event (same day)

- [ ] Thank attendees verbally and in chat
- [ ] Share feedback survey link
- [ ] Post event photos to social media
- [ ] Save recording and chat logs
- [ ] Collect any lost-and-found items (in-person)
- [ ] Quick debrief with team (5 min: what went well, what to fix)
