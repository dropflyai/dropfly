# Naming Brief — Template

## Purpose

This template structures the inputs required for a naming project. Naming is one of the highest-stakes brand decisions: a name is the most frequently used brand element, the hardest to change, and the most constrained by legal, linguistic, and digital availability factors. A rigorous brief prevents the naming process from devolving into subjective preference debates. The brief establishes objective criteria against which all name candidates are evaluated.

## 3. Naming Criteria

### Name Type Preference

| Type | Definition | Example | Consideration |
|------|-----------|---------|---------------|
| Descriptive | Describes what the entity does | "General Motors" | Easy to understand; hard to trademark |
| Suggestive | Suggests a quality or benefit | "Pinterest" (pin + interest) | Balances meaning and ownerability |
| Abstract / Coined | Invented word with no inherent meaning | "Spotify," "Xerox" | Highly ownable; requires education |
| Associative | Real word used in unexpected context | "Apple," "Amazon" | Memorable; meaning borrowed from context |
| Acronym | Initials of longer name | "IBM," "BMW" | Efficient; lacks inherent meaning |
| Founder/Personal | Based on person's name | "Tesla," "Chanel" | Personal equity; succession challenge |

## 5. Evaluation Criteria

### Scoring Rubric
All name candidates will be scored on these criteria (1-5 each):

| Criterion | Weight | Definition |
|-----------|--------|-----------|
| Strategic fit | [30%] | Does the name support the positioning and evoke desired attributes? |
| Distinctiveness | [20%] | Does the name stand out from competitors and category conventions? |
| Memorability | [15%] | Is the name easy to recall after exposure? |
| Pronounceability | [10%] | Can the target audience say it correctly on first attempt? |
| Spellability | [5%] | Can the target audience spell it after hearing it? |
| Trademark viability | [10%] | Is the name likely to clear trademark screening? |
| Domain/handle availability | [5%] | Is the .com (or acceptable alternative) and social handles available? |
| Linguistic safety | [5%] | Does the name avoid negative connotations in key languages? |

### Minimum Thresholds
- Weighted score must be >= [3.5] out of 5.0 to proceed to shortlist
- Trademark viability must score >= 4 (no candidates with clear conflicts)
- Linguistic safety must score >= 4 (no candidates with negative connotations in any target language)

## 6. Process and Timeline

### Naming Process

| Phase | Activities | Duration | Output |
|-------|-----------|----------|--------|
| 1. Brief | Complete and approve this brief | [X weeks] | Approved brief |
| 2. Generation | Brainstorm, explore territories, generate 150-300+ candidates | [X weeks] | Long list |
| 3. Screening | Linguistic check, preliminary trademark, domain check | [X weeks] | Screened list (30-50) |
| 4. Shortlist | Apply scoring rubric, stakeholder review | [X weeks] | Shortlist (5-8) |
| 5. Testing | Qualitative and/or quantitative testing with target audience | [X weeks] | Tested shortlist (2-3) |
| 6. Legal clearance | Full trademark search and clearance | [X weeks] | Cleared finalist(s) |
| 7. Selection | Final decision by Decision Maker | [X days] | Selected name |
| 8. Registration | Trademark filing, domain acquisition, social handles | [X weeks] | Registered name |

### Decision Process
- **Decision Maker:** [Name] makes the final call
- **Input providers:** [Names] provide scored feedback using the rubric
- **Rule:** If consensus is not reached on the shortlist, the Decision Maker decides. Naming by committee produces mediocre names.

## Usage Notes

- The naming brief must be approved BEFORE any name generation begins
- Do not skip linguistic screening; negative connotations in even one target market can be fatal
- Trademark clearance is expensive; screen progressively (preliminary check on many, full search on few)
- The Decision Maker should be a single individual, not a committee
- Reference `02_strategy/` for brand architecture and positioning frameworks
- Archive completed briefs and naming outcomes for institutional memory
