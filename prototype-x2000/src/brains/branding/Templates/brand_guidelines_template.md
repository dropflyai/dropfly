# Brand Guidelines — Template

## Purpose

This template provides the structure for a comprehensive brand guidelines document. Brand guidelines codify the brand identity system so that anyone creating brand touchpoints — internal teams, external agencies, partners — can do so consistently and correctly. The guidelines serve as both a reference document and an educational tool, explaining not just what to do but why.

## Usage Notes

- This template should be adapted to the brand's specific needs; not all sections apply to every brand

- Update at least annually, or whenever the brand evolves
