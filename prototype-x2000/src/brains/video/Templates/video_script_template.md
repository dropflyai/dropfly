# Video Script — Template

## Purpose

This template provides a structured framework for writing video scripts. A well-written script is the single largest determinant of video quality — no amount of production value compensates for a poorly structured script. This template enforces the hook-body-CTA structure proven to maximize viewer retention and drive desired actions across all video formats and platforms.

## Script Format Guide

### Column Format

Each script section uses a three-column format:

| VISUAL | AUDIO | ON-SCREEN TEXT |
|--------|-------|---------------|
| What the viewer sees (camera angle, B-roll, graphics, animation, screen recording) | What the viewer hears (narration, dialogue, music, SFX) | Text overlays, lower thirds, captions, titles |

### Timing Convention
- Mark estimated timestamps at each section: `[0:00]`, `[0:30]`, `[1:00]`
- Narration timing: ~150 words per minute (conversational pace)
- Allow 2-3 seconds for visual transitions
- Allow 3-5 seconds for graphics to be read

## Pre-Script Checklist

Before writing, confirm:

- [ ] Video brief is approved and available
- [ ] Target audience is defined (knowledge level, viewing context)
- [ ] Core message is articulated in one sentence
- [ ] Platform requirements are known (duration, aspect ratio, sound context)
- [ ] Call to action is defined (what should the viewer do?)
- [ ] Key data points, quotes, or demonstrations are identified
- [ ] Brand voice guidelines have been reviewed

## Short-Form Script Addendum (Under 60 seconds)

For TikTok, Reels, Shorts, and other short-form formats, use this condensed structure:

| Timestamp | Element | Duration |
|-----------|---------|----------|
| 0:00-0:03 | **HOOK:** Pattern interrupt or bold statement | 3 seconds |
| 0:03-0:10 | **CONTEXT:** Why should the viewer care? | 7 seconds |
| 0:10-0:45 | **VALUE:** The actual content (tip, story, demo) | 35 seconds |
| 0:45-0:55 | **PAYOFF:** Resolution, result, or surprise | 10 seconds |
| 0:55-0:60 | **CTA:** Follow, comment, or save | 5 seconds |

**Short-Form Script Example:**

| VISUAL | AUDIO | ON-SCREEN TEXT |
|--------|-------|---------------|
| [Close-up, high energy] | "Stop doing [X]. Here's what to do instead." | "STOP doing [X]" |
| [Cut to demonstration] | [Quick explanation of the alternative approach] | [Key steps as text] |
| [Result/evidence] | "And the result? [impressive outcome]" | [Result stat/image] |
| [Direct to camera] | "Save this for later. Follow for more." | "SAVE + FOLLOW" |

## Script Review Checklist

Before submitting for approval, verify:

- [ ] Hook captures attention within first 3-5 seconds
- [ ] Core message from brief is clearly communicated
- [ ] Script is written conversationally (read it aloud — does it sound natural?)
- [ ] Every section delivers distinct value (no padding or filler)
- [ ] On-screen text is planned for sound-off viewing
- [ ] CTA is specific, actionable, and supported by a reason
- [ ] Estimated duration matches the brief target (± 10%)
- [ ] Brand voice guidelines are followed
- [ ] Technical accuracy verified with SME (if applicable)
- [ ] Visual notes are specific enough for the production team

## Usage Notes

- One script per video (do not combine multiple videos in one script)
- Read the script aloud at conversational pace to verify timing
- Have someone unfamiliar with the topic read the script — if they cannot follow, simplify
- Keep narration at 150 words/minute maximum — rushing reduces comprehension
- Visual notes should be specific enough for someone else to produce the video from the script alone
- Archive approved scripts with their performance data for institutional learning
