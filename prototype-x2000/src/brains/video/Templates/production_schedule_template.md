# Production Schedule — Template

## Purpose

This template structures the complete production schedule for a video project, from pre-production through final delivery. It ensures all tasks are sequenced correctly, dependencies are mapped, owners are assigned, and deadlines are realistic. The production schedule is the operational complement to the video brief — while the brief defines what to create, the schedule defines when and how.

## Usage Notes

- Adjust timeline based on project complexity (simple = compress; premium = extend)
- All gate tasks (Brief Approved, Script Approved, Final Approved) are blocking — no downstream work begins until gate passes
- Review turnaround times should be agreed upon and enforced
- This schedule should be maintained in a project management tool (Asana, Monday, Notion) for real-time tracking
- Post-project, archive this schedule with performance data for future production planning
