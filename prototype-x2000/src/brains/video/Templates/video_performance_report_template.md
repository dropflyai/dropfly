# Video Performance Report — Template

## Purpose

This template structures the periodic review of video content performance. It enables data-driven decisions about content strategy, production investment, and distribution optimization. The report should be completed weekly (abbreviated version) and monthly (full version), with quarterly deep-dives for strategic planning.

## Usage Notes

- Weekly report: Complete sections 1 (summary), 2 (platform performance), and 3 (content performance) only
- Monthly report: Complete all sections
- Quarterly report: Complete all sections plus deeper competitive analysis and strategic review
- All metrics should be pulled from platform-native analytics tools for accuracy
- Automated dashboards (Databox, Geckoboard, custom) should supplement but not replace this narrative report
- Archive reports for longitudinal trend analysis
- Share report within 3 business days of period end
