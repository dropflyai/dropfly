## 8. PRE-EVENT PREPARATION

### 1 Week Before

- [ ] All speakers confirmed and briefed
- [ ] Slides/materials reviewed
- [ ] Platform tested (audio, video, screen share)
- [ ] Registration list downloaded and reviewed
- [ ] Promotional posts scheduled
- [ ] Recording setup confirmed
- [ ] Facilitator notes prepared

### 1 Day Before

- [ ] Final reminder sent to registrants
- [ ] Speaker tech check completed
- [ ] Backup plan confirmed
- [ ] Agenda printed/accessible
- [ ] Chat prompts prepared
- [ ] Q&A seed questions prepared (in case of silence)

### 30 Minutes Before

- [ ] Platform opened and tested
- [ ] Waiting room configured (if applicable)
- [ ] Recording started (test)
- [ ] Welcome slide or message ready
- [ ] Co-host/moderator connected
- [ ] Emergency contact available

---

## 9. POST-EVENT FOLLOW-UP

### Within 24 Hours

- [ ] Thank-you message to attendees (with recording link if applicable)
- [ ] Thank-you message to speakers
- [ ] Recording processed and shared
- [ ] Key highlights posted in community
- [ ] Feedback survey sent

### Within 1 Week

- [ ] Feedback survey results reviewed
- [ ] Event metrics compiled
- [ ] Retrospective with team
- [ ] Content assets created (blog post, clips, etc.)
- [ ] Next event promoted

**This template ensures consistent, high-quality event execution. Adapt it

and post-event follow-up.**
