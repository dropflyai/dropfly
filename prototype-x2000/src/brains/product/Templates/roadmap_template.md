## Instructions for Quarterly Roadmap Planning

```
PRE-PLANNING (2 weeks before)
1. Review outcomes from current quarter (did NOW items hit targets?)
2. Update NEXT items based on discovery progress
3. Refresh evidence for all items (new interviews, data, competitive intel)
4. Gather input from Sales, CS, Support, Engineering

PLANNING SESSION (1 day)
1. Retrospective: what did we learn this quarter? (1 hour)
2. Strategy review: is the strategy still correct? (1 hour)
3. NEXT -> NOW promotion: which NEXT items are ready? (2 hours)
4. NEW items: any new opportunities to add? (1 hour)
5. Capacity allocation: does the team have capacity for the plan? (1 hour)
6. Tradeoffs: what are we explicitly not doing? (30 min)
7. Communication plan: who needs to know and how? (30 min)

POST-PLANNING (1 week after)
1. Finalize roadmap document
2. Communicate to all stakeholders
3. Update audience-specific views
4. Set up monitoring for NOW item metrics
```
