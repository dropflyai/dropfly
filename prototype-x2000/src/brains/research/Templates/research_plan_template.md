| Risk | Probability | Impact | Mitigation |
|------|-----------|--------|-----------|
| Insufficient recruitment | | | Over-recruit, multiple channels |
| Response bias | | | Careful question design, mixed methods |
| Low response rate | | | Increase incentive, follow-up reminders |
| Findings inconclusive | | | Ensure sufficient sample size, clear hypotheses |
| Stakeholder disagreement with findings | | | Involve stakeholders in planning, transparent methodology |
| Timeline slippage | | | Buffer time in schedule, parallel workstreams |
