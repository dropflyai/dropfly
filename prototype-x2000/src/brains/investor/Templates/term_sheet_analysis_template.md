# Term Sheet Analysis — Template

## Purpose

This template provides a structured framework for analyzing a venture capital term sheet. Every term has economic or control implications; this template ensures no term is overlooked, each is benchmarked against market standards, and the overall package is evaluated holistically. The template should be completed in collaboration with experienced legal counsel — founders should never negotiate term sheets alone.

## Usage Notes

- This analysis should be completed BEFORE responding to the term sheet
- Always review with experienced startup legal counsel
- Do not negotiate individual terms in isolation — consider the package holistically
- Reference `04_term_sheets/` module files for detailed term-by-term guidance
- Archive completed analyses for institutional memory and pattern recognition
- The term sheet is non-binding (except no-shop); the real negotiation happens in definitive documents
