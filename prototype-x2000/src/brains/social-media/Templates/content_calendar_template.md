## Platform-Specific Content Specifications

### Instagram

| Format | Dimensions | Duration | File Type | Notes |
|--------|-----------|----------|-----------|-------|
| Feed (Square) | 1080x1080 | N/A | JPG/PNG | Grid aesthetic consideration |
| Feed (Portrait) | 1080x1350 | N/A | JPG/PNG | Maximum feed real estate |
| Carousel | 1080x1080 or 1080x1350 | N/A | JPG/PNG (up to 10) | Consistent dimensions across slides |
| Reel | 1080x1920 | 15-90s | MP4 (<4GB) | Cover image: 1080x1920 |
| Story | 1080x1920 | <60s (video) | JPG/PNG/MP4 | Safe zones for text |

### LinkedIn

| Format | Dimensions | Duration | File Type | Notes |
|--------|-----------|----------|-----------|-------|
| Image | 1200x628 or 1080x1080 | N/A | JPG/PNG | Professional quality |
| Document | 1080x1350 (portrait pages) | N/A | PDF | 8-12 pages optimal |
| Video | 1920x1080 or 1080x1080 | <10 min | MP4 | Subtitles required |

### TikTok

| Format | Dimensions | Duration | File Type | Notes |
|--------|-----------|----------|-----------|-------|
| Video | 1080x1920 | 15-180s | MP4 | Native aesthetic preferred |

### Facebook

| Format | Dimensions | Duration | File Type | Notes |
|--------|-----------|----------|-----------|-------|
| Image | 1200x630 or 1080x1080 | N/A | JPG/PNG | <20% text overlay |
| Video | 1080x1080 or 1920x1080 | <240 min | MP4 | Square performs best in feed |
| Reel | 1080x1920 | 15-90s | MP4 | Cross-post from Instagram |
