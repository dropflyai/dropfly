# Board Meeting Template

## Instructions

This template is used by the CEO Brain to prepare for and conduct board
meetings. Complete all sections before the meeting. Send the board deck
to directors 48+ hours in advance. The goal is productive strategic
discussion, not status theater.

---

## Board Meeting Agenda

### Segment 1: Financial Review (45 minutes)

```
PRESENTER: CEO + CFO
FORMAT: Walkthrough of financial package

DISCUSSION POINTS:
1. Revenue performance vs. plan -- what drove variance?
2. Expense management -- any unexpected costs?
3. Cash position and runway -- any concerns?
4. Unit economics trends -- improving or degrading?

QUESTIONS TO ANTICIPATE:
- Why did [metric] miss plan?
- What are you doing about [concerning trend]?
- When do you expect to reach [milestone]?
```

### Segment 2: Business Update (45 minutes)

```
PRESENTER: CEO

PRODUCT UPDATE:
- What shipped this quarter
- What we learned from customers
- Product roadmap priorities for next quarter
- Key product risks or decisions

CUSTOMER UPDATE:
- Notable wins and losses
- Customer sentiment (NPS, feedback themes)
- Retention trends and churn analysis
- Top customer requests

TEAM UPDATE:
- Key hires and departures
- Team health (engagement, retention)
- Organizational changes
- Upcoming hiring needs

COMPETITIVE UPDATE:
- Notable competitor moves
- Market trends
- Our competitive response
```

### Segment 3: Strategic Discussion (60 minutes)

```
FORMAT: Discussion, not presentation. CEO frames the question;
board provides input.

STRATEGIC TOPIC 1: [Topic]
+--------------------------------------------------+
| CONTEXT: [Why this topic matters now]            |
| OPTIONS:                                          |
| A. [Option A] -- Pros: [X] / Cons: [X]         |
| B. [Option B] -- Pros: [X] / Cons: [X]         |
| C. [Option C] -- Pros: [X] / Cons: [X]         |
| CEO RECOMMENDATION: [Which option and why]       |
| BOARD INPUT REQUESTED: [Specific question]       |
+--------------------------------------------------+

STRATEGIC TOPIC 2: [Topic]
+--------------------------------------------------+
| [Same format as above]                           |
+--------------------------------------------------+
```

### Segment 4: Closed Session (30 minutes)

```
FORMAT: Board members only (or CEO excused for part)

TOPICS:
- CEO performance and feedback
- Compensation matters
- Sensitive personnel issues
- Board governance
- Anything directors want to discuss privately
```

## Board Meeting Preparation Timeline

```
T-4 WEEKS: Begin drafting CEO letter and financial package
T-3 WEEKS: Identify strategic discussion topics; gather input
T-2 WEEKS: Complete board deck draft; internal review
T-1 WEEK:  Finalize deck; pre-brief directors individually
            on sensitive topics
T-48 HOURS: Send board package to all directors
T-0:        Board meeting
T+48 HOURS: Send minutes and action items
T+1 WEEK:   Begin working on action items
```

---

## Board Meeting Anti-Patterns

| Anti-Pattern | Description | Fix |
|-------------|------------|-----|
| Status theater | Entire meeting is reporting, no discussion | 60%+ time on strategic discussion |
| Surprises | Board learns bad news in the meeting | Pre-brief individually |
| No asks | CEO does not ask board for help | Always include 2-3 specific asks |
| No follow-up | Minutes and actions are not tracked | Send within 48 hours, track to completion |
| Over-prepared | 100-slide deck that takes 4 hours | Max 15 slides; pre-read does the heavy lifting |
| Under-prepared | CEO wings it | Preparation starts 4 weeks before |
