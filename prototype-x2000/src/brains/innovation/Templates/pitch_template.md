## Pitch Preparation Checklist

### Before the Pitch

- [ ] Know your audience (what they care about, what their objections will be)
- [ ] Practice the pitch within time limit (3 rehearsals minimum)
- [ ] Prepare for the top 5 likely objections
- [ ] Have backup slides for detailed questions
- [ ] Know your numbers cold (financials, metrics, timeline)
- [ ] Have a clear, specific ask (not vague)
- [ ] Have a demo or visual ready (seeing is believing)

### Common Objections and Responses

| Objection | Prepared Response |
|-----------|------------------|
| "What if it fails?" | |
| "How does this fit with our strategy?" | |
| "Do we have the resources?" | |
| "What about competitor X?" | |
| "Can you prove customer demand?" | |
| "What is the ROI?" | |
| "Why should we do this now?" | |

### Post-Pitch Follow-Up

| Action | Timeline |
|--------|----------|
| Send follow-up email with key points | Within 24 hours |
| Address any unanswered questions | Within 48 hours |
| Provide additional data requested | Within 1 week |
| Schedule next check-in | Before leaving the room |

---

**The best internal pitches are clear, evidence-based, and honest about uncertainty. Do not oversell -- executives respect intellectual honesty. Show what you know, acknowledge what you do not, and explain how you will learn what you need to learn.**
