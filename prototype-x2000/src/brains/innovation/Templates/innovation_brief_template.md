| Feasibility | Technical and business feasibility confirmed | | [ ] Go [ ] No-Go [ ] Pivot | |
| Prototype | Prototype tested with real users | | [ ] Go [ ] No-Go [ ] Pivot | |
| Pilot | Pilot results meet success criteria | | [ ] Go [ ] No-Go [ ] Pivot | |
| Scale | Pilot validated, unit economics viable | | [ ] Go [ ] No-Go [ ] Pivot | |

| Review | Cadence | Attendees |

| Stage gate review | Per gate | Leadership team |

**An innovation brief is a living document. Update it as you learn. The best innovation briefs start with strong problem definition and evolve through rapid experimentation. Kill fast when the evidence says kill. Double down when the evidence says go.**
