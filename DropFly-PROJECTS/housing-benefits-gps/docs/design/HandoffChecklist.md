## Implementation Notes

### Accessibility Requirements

- [ ] All touch targets ≥ 44px
- [ ] All text contrast ≥ 4.5:1
- [ ] Focus states on all interactive elements
- [ ] Screen reader labels on icons
- [ ] Form field labels properly associated
- [ ] Error messages announced to screen readers
- [ ] Progress announced on step changes

### Performance Requirements

- [ ] First contentful paint < 2s
- [ ] Time to interactive < 3s
- [ ] Smooth 60fps animations
- [ ] Image optimization (thumbnails, lazy loading)
- [ ] Offline support for saved data (nice-to-have)

### Security Requirements

- [ ] HTTPS everywhere
- [ ] SMS codes expire in 5 minutes
- [ ] Session tokens with appropriate expiry
- [ ] No PII in URLs or logs
- [ ] Encrypted storage for sensitive data
- [ ] Rate limiting on auth endpoints

## Content to Implement

### Tile Content (LA County MVP)

| Tile | Title | Programs |
|------|-------|----------|
| rent_help | "Stop Eviction & Get Rent Help" | LA County Rent Relief, Stay Housed LA |
| food | "Get Food Money (EBT)" | CalFresh/SNAP |
| cash | "Monthly Cash" | CalWORKs, General Relief |
| health | "Health Coverage" | Medi-Cal, Covered California |
| childcare | "Child Care Help" | CalWORKs Stage 1/2/3, Head Start |
| housing_longterm | "Housing Voucher" | Section 8 waitlist, public housing |

### Wizard Content per Tile

Each tile needs:
- 1 Info step (educational content)
- 1 Confirmation step (verify user data)
- 1 Document upload step (checklist)
- 1 External action step (link to application)
- 1 Status check step
- 1 Success step

Total: ~36 wizard steps to write content for

### Fallback Content

- 211 information for unsupported jurisdictions
- Legal aid referral content
- Homeless prevention resource content
