### Spacing Scale

| Token | Value | Use |
|-------|-------|-----|
| `xs` | 4px | Icon to text gaps |
| `sm` | 8px | Related elements |
| `md` | 16px | Between components |
| `lg` | 24px | Section spacing |
| `xl` | 32px | Major sections |
| `2xl` | 48px | Page-level breathing room |

## Accessibility Requirements

### Contrast Ratios

All text/background combinations meet WCAG 2.1 AA standards:
- Normal text: minimum 4.5:1 contrast ratio
- Large text (18px+): minimum 3:1 contrast ratio

### Touch Targets

- Minimum touch target: 44px × 44px
- Recommended: 48px × 48px or larger
- Buttons: 52px minimum height

### Focus States

```css
:focus-visible {
  outline: 2px solid var(--color-primary);
  outline-offset: 2px;
}
```
