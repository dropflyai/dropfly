## 7. UI States Reference

### Global States

| State | Implementation |
|-------|----------------|
| **Loading** | Skeleton loaders for content, spinners for buttons |
| **Empty** | Friendly message + primary action CTA |
| **Error** | Toast notification + retry option |
| **Success** | Checkmark animation + confirmation message |
| **Offline** | Banner: "You're offline. Some features unavailable." |

### Per-Screen States

| Screen | Loading | Empty | Error | Success |
|--------|---------|-------|-------|---------|
| Landing | Button spinner | N/A | Toast | Redirect |
| Sign Up | Button spinner | N/A | Inline + shake | Auto-advance |
| Intake | Progress pulse | N/A | Inline below field | Auto-advance |
| Home | Skeleton tiles | "Let's build your plan" | Toast + retry | Tile updates |
| Wizard | Step skeleton | N/A | Inline / toast | Checkmark anim |
| Doc Upload | Progress % | "No docs yet" | "Upload failed" | Checkmark |

### Animation Specs

| Animation | Duration | Easing |
|-----------|----------|--------|
| Page transition | 250ms | ease-out |
| Button press | 150ms | ease-in-out |
| Progress bar | 250ms | ease-out |
| Checkmark draw | 400ms | ease-out |
| Toast slide | 250ms | ease-out |
| Modal fade | 200ms | ease-out |
| Skeleton pulse | 1.5s | infinite ease-in-out |
